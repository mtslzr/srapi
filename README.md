# SRAPI

SRAPI is a Go-based API for the Sports Reference network of websites.

## Installation

### Clone Repo

```bash
git clone git@github.com:mtslzr/srapi
cd srapi
```
### Build

```bash
go build
```

## Usage

```bash
./srapi
Starting server on localhost:5000...
```
